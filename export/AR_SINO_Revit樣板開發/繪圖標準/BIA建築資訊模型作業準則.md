一般通則

壹、檔案命名原則

專案檔名命名原則：

專案名(編號)-模型類別 $\left( \mathrm { A R } \cdot \mathrm { S } \cdot \mathrm { E } \cdot \mathrm { S } ^ { \ldots } \right.$ 等)-棟別-階段 $\mathrm { S D } \setminus \mathrm { D D } \cdots$ 等)-模型版本-日期+版本儲存原因

範例檻-AR-SD WT
範例槛-AR-SD-V1-0731例行存檔
範例槛-AR-SD-V2-0806地下唇變更
範例檻-AR-SD-V3-0816標準層變更

避免檔案連結失效，主作業檔為專案名-類別-階段的 RVT 檔為作業檔，附上模型版本及日期備註的為設計變更及設計節點紀錄檔

參、標註

一、尺寸

尺寸標註類型：

短斜線標記統一使用「實圓點 1.5mm」  
文字大小 $: 1 / 2 0 0$ 以上 3.0mm，1/200\~400 2.5mm，1/400 以下 2.0mm字型統一使用微軟正黑體

二、門窗

門編號以圓形圖示標註，上編號為門編號以 D、DW 開頭，下編號為防火時效 上下編號字體大小 1/200 以上 3.5mm，1/200\~400 3mm，1/400 以下 2.5mm，統一使用微軟正黑體窗編號以圓形圖示標註，編號為窗編號以 W 開頭，字體大小與字體型式與門標註相同

第二章 基本設計

第一節 模型完成事項

壹、空間計畫：房間-房間名稱標註(標號、名稱、面積)，尺寸標註區域-視圖依照使用目的分類，建照、空間區劃等，空間名稱依照目的、建照、建築技術規則內容命定，居室、梯廳、安全梯等

貳、明細表：房間明細表，檢討業主需求。區域明細表，檢討建照、樓地板面積、業主需求其他明細表，計算數量如車位數，依需求建置

第二節 衝突干涉檢討事項

針對法規檢討衝突干涉

1. 步行距離，70cm 避開障礙物至梯間的距離，重複步行距離不可超過 35cm  
2. 防火區劃，倉儲空間 $3 0 0 0 \mathrm { m } ^ { 2 }$ 以下，辦公室 $1 5 0 0 \mathrm { m } ^ { 2 }$   
3. 無障礙廁所，每三層樓設置一間  
4. 至少一座無障礙樓梯  
5. 車道口梁下淨高 2.1M  
6. 衛生設備數量  
7. 女兒牆及欄杆扶手限制  
依各棟需求檢討其他項目
